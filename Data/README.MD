# Dataset Information

**Full Dataset:**  
Available on Kaggle: [Aircraft Noise Report SFO](https://www.kaggle.com/datasets/porhayrouen/aircraft-noise-report-sfo)

**Description:**  
The full dataset contains aircraft noise complaint reports from 2018 to the present.  
It includes details such as complaint time, reporter location, aircraft altitude, type, and operation type.

**Sample File Included:**  
This repository only includes a small sample file (`sample_aircraft_noise.csv`) with the first 100 records for demonstration purposes.
